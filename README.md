# Spring-Board-Project 

## 프로젝트 소개
휴식 후 기존에 배운 내용을 복습 및 
스프링 시큐리티를 적용하여 로그인 기반의 
텍스트 게시판 제작


## 개발기간 
24.03.1주차~24.03.3주차

### 개발환경

- Java 11
- IDE : IntelliJ
- FrameWork : Spring
- DataBase : Oracle 21c
- ORM : Mybatis

  배포 X
